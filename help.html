Справка по системе мониторинга

Общая информация

Система мониторинга нефтетанкеров и нефтевышек предоставляет реальные данные о перемещении и состоянии объектов нефтедобычи по всему миру.

Работа с картой

Нефтетанкеры

Отображаются синими значками. Наведите курсор для просмотра информации о танкере.

Нефтевышки

Отображаются желтыми значками. Нажмите для получения детальной информации.

Фильтрация

Используйте панель фильтров для отображения только нужных типов объектов.

Статусы объектов

НОРМАЛЬНЫЙ

Объект работает в штатном режиме

ПРЕДУПРЕЖДЕНИЕ

Обнаружены незначительные отклонения

ТРЕВОГА

Требуется вмешательство персонала

Новостная панель

В правой части экрана расположена панель новостей нефтяного бизнеса:

  • Кнопка "Вверх" - предыдущая новость
  • Кнопка "Вниз" - следующая новость
  • Кнопка "Динамик" - включение/выключение озвучки
  • Кнопка "Свернуть" - скрыть/показать новости

Статистическая панель

Отображает ключевые показатели системы в реальном времени:

  • Количество активных объектов
  • Объем суточной добычи
  • Количество объектов в пути
  • Текущие тревоги

Горячие клавиши

F1 Открыть справку
F2 Обновить данные
Ctrl + N Следующая новость
Ctrl + P Предыдущая новость
Ctrl + M Открыть карту
Esc Закрыть окно

Техническая поддержка: Для получения дополнительной помощи обратитесь в службу поддержки по телефону, указанному на странице "Контакты".
