# 🛢️ **НЕФТЕКОНТРОЛЬ-94**  
### *Система мониторинга нефтедобычи в стиле 90-х*

> 🚀 **Краткое описание**  
> Веб-интерфейс в ретро-техно стиле для отслеживания глобальных нефтетанкеров и вышек. Включает фильтрацию по континентам, новостную ленту с TTS-озвучкой, статистику и цветовую индикацию статусов.  
> *Дизайн вдохновлён терминалами промышленных АСУ ТП и кинематографом 90-х (с отсылками к «Матрице» и Warhammer 40k техно-эстетике).*

---

## 📌 Основные функции

### 1. **Мониторинг объектов**

- 🌍 **Карта мира** с векторной разметкой:
  - Танкеры → синие треугольники (направление движения отображается остриём)
  - Нефтяные вышки → жёлтые квадраты с пульсирующим свечением
- 🗺 **Фильтр по континентам**: Европа, Азия, Африка, Америка, Австралия
- 🚦 **Статусы объектов**:
  | Цвет       | Статус        | Анимация           |
  |------------|---------------|--------------------|
  | `#00ff00`  | Нормальный    | —                  |
  | `#ffff00`  | Предупреждение| Медленное мерцание |
  | `#ff0000`  | Тревога       | Быстрое мигание + звуковой сигнал |
- ℹ️ **Всплывающие подсказки** (по `hover`/`tap`) с деталями: название, ID, груз (для танкеров), текущая глубина/добыча (для вышек), GPS-координаты.

---

### 2. **Новостная панель**

📰 Лента обновлений из нефтяного сектора (API-поток в реальном времени).

- 🔄 Навигация:  
  `▲` — предыдущая новость | `▼` — следующая новость  
- 🔊 **TTS-озвучка** (Web Speech API):  
  `🔊` — вкл/выкл синтезатора (голос: *«RoboVoice Mk.III»*, темп 110%)  
- 📱 Управление:
  - `↔` — свернуть/развернуть панель  
  - `⏯` — открыть полный текст в модальном окне (`modal_news.html`)

---

### 3. **Статистика**

| Показатель                     | Значение (пример)       |
|-------------------------------|-------------------------|
| Активные танкеры              | **142**                 |
| Работающие вышки              | **87**                  |
| Суточная добыча               | **12.4 млн баррелей**   |
| Танкеры в пути                | **53**                  |
| Текущие тревоги / предупреждения | **3** / **7**        |

*Данные обновляются раз в 30 секунд через `fetch()` → `localStorage` кэширование.*
